# Análise de Dados e Dashboard Interativo no Excel

### Sobre o Projeto

O projeto consiste no tratamento, organização e análise exploratória de um dataset contendo informações de vendas de uma loja de bicicletas.

O objetivo foi transformar os dados brutos em informações organizadas e visualmente interpretáveis, gerando um dashboard interativo no Excel, possibilitando a extração de informações valiosas.

O projeto envolveu limpeza de dados, padronização de variáveis, categorização de informações e criação de visualizações dinâmicas para análise mais estratégica.

#### Ferramentas Utilizadas

Microsoft Excel

Tabelas Dinâmicas

Segmentação de Dados

Fórmulas (SE, tratamento de texto e categorização)

Gráficos Dinâmicos

---

### Etapas do Projeto

1. Limpeza e Tratamento de Dados

Remoção de duplicidades

Localização e substituição de dados para maior clareza

Tradução e padronização de categorias (ex: ocupação e região)

Organização das variáveis para melhorar a consistência da base

2. Transformação e Categorização

Criação de fórmulas utilizando SE para categorização de idade em faixas

Agrupamento de informações para gerar visualizações mais limpas

Padronização de categorias para facilitar análise estatística

3. Análise Exploratória

Construção de Tabelas Dinâmicas

Cruzamento de variáveis como Região e Ocupação

Análise de distribuição por categorias

4. Construção do Dashboard

Desenvolvimento de gráficos dinâmicos

Criação de filtros interativos (Segmentação de Dados)

Organização visual com legendas e estrutura intuitiva

Dashboard completo permitindo análise dinâmica e detalhada

---

#### Dashboard Interativo

O dashboard permite:

Filtrar dados por região, genero, escolaridade e estado civil

Visualizar distribuições por faixa etária

Analisar padrões de forma dinâmica

---
A estrutura foi pensada para proporcionar clareza visual e facilitar a tomada de decisão com base nos dados.
